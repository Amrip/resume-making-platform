# Resume Maker Platform

This is a platform for creating and sharing resumes.

## Technologies Used

- React18
- Electron27
- node21
- webpack5
- sass
- eslint
- prettier

## Features

- [ ] Create a resume
- [ ] Edit a resume
- [ ] Share a resume
- [ ] View other users' resumes
- [ ] Search for resumes
- [ ] Download a resume as a PDF

## Status

This project is currently in development.
